

The [[Git Log]] command isn't only useful for your _local_ [[repositorio]]. You can log the commits of a [[Remote]] repo as well!

```bash
git log [<remote>/<branch>]
```

For example, if you wanted to see the [[Commit]]s of a [[Branch]] named `primeagen` from a remote named `origin` you would run:

```bash
git log origin/primeagen
```
