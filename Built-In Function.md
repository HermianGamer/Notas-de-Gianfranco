A built-in [[Function]] is a predefined set of instructions that comes standard with a programming language, application, or tool. These functions are an intrinsic part of the core [[enviroment]], meaning they are ready to use immediately without requiring additional installation or the manual definition of code.

Key Characteristics

- **Ready-to-Use:** They live in the language's global namespace and can be called directly, saving time and reducing the need to write complex code from scratch.
- **Standardization:** Because they are part of the core language, they are standardized, well-documented, and highly optimized for performance.
- **Common Tasks:** They are designed to handle repetitive and universal operations like mathematical calculations, string manipulation, and input/output handling.
