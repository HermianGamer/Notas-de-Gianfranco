
Saber si un elemento está en una [[Lista]] o en un [[Diccionario]]. Para esto se una `in`

Lista:
```py
fruits = ["apple", "orange", "banana"]
print("banana" in fruits)
# Prints: True
```

Diccionario:
If you're unsure whether a key exists in a dictionary, use the `in` keyword.
```py
cars = {
    "ford": "f150",
    "toyota": "camry"
}

print("ford" in cars)
# Prints: True

print("gmc" in cars)
# Prints: False
```
