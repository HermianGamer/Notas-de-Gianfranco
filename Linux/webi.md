  
There is one more [[Package Manager]] (actually more of an anti-package manager) I want to introduce you to: [Webi](https://webinstall.dev/).

Webi lets you install [[CLI]] tools directly from the web, with no need for a local command line tool like `apt` or `brew`. You don't need to install webi itself at all, instead, you just run a [[Shell]] command that downloads and runs the tool's official installer script.
