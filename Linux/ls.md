commando de [[Terminal]] para ver todos los  [[File]]s dentro del [[Directory]] actual.

```bash
(base) gianfrancoscarpati@MacBook-Air-de-Gianfranco ~ % ls
```

tambien puedes ver el `ls` de otro directorio si es que lo especificas:
```bash
ls worldbanc
```

The `ls` command has a `-l` option (lowercase "L") that will print out the permissions of each file and directory in long format.
```bash
ls -l private
```
The `ls` command has a `-a` flag that will show hidden files:

```bash
ls -a ~
```
