Now, some developers do their work inside a terminal and use a terminal editor like [Neovim](https://neovim.io/), but a lot of developers, myself included, prefer to use a hybrid /terminal experience.

Enter [VS Code](https://code.visualstudio.com/). It's a fairly lightweight editor that's easy to use for beginners, but powerful enough for professionals. VS Code (not to be confused with [Visual Studio](https://visualstudio.microsoft.com/)) is the [most popular](https://survey.stackoverflow.co/2025/technology#1-dev-id-es) code editor among developers today.

Some other cool [[GUI]] editors include:

- [Zed](https://zed.dev/) (I actually switched to Zed from VS Code recently)
- [Cursor](https://www.cursor.com/)
- [IntelliJ IDEA](https://www.jetbrains.com/idea/)

And some of the most popular terminal editors are:

- [Neovim](https://neovim.io/) o [[nvim]]
- [Vim](https://www.vim.org/)
- [Emacs](https://www.gnu.org/software/emacs/)
