A [`README.md`](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes) is a [markdown](https://www.markdownguide.org/getting-started/) documentation file that's supposed to explain what your [[project]] is and how to run it.

And if you're a _real developer_™ then it's always out of date and doesn't provide enough info.
